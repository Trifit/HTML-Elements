Modal dialog

In order to implement this widget you'll need to create a clickable element (link or button) with a class named "js-open-dialog" in your html and include the script dialog.js in your html

The js script will create a div with the modal inmideatly bellow the button.

Options

  • Include class to convert modal into a tooltip directly attached to the clickable element
